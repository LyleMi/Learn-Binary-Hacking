MSVC
========================================
Microsoft Visual C++是微软公司的C++开发工具，具有集成开发环境，可提供编辑C语言，C++以及C++/CLI等编程语言。

MSVC的特点有：支持增量编译和增量链接；支持C++规范较快等。
